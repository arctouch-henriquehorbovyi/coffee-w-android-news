### **Dica da Semana**

Quer monitorar a performance da sua aplicação com baixo overhead? Use o **Jetpack Macrobenchmark** para medir tempo de inicialização e transições críticas:

```
@ExperimentalMetricApi
@Composable
fun BenchmarkApp() {
    val metrics = rememberMacroMetrics()
    Box(
        modifier = Modifier
            .benchmarkable(metrics)
            .fillMaxSize()
    ) {
        Text("Hello, World!")
    }
}
```
